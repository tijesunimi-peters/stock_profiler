# Project constraints

- **SEC financials only.** All data is derived from SEC filings (10-K/10-Q XBRL
  facts, 8-K events, Forms 3/4/5 insider transactions, S-1/registration, etc.).
  **No market data** (prices, market cap, P/E, EV/EBITDA, dividend/buyback yield,
  valuation multiples, returns). Do not add any metric that requires market
  pricing until the user says market data is available.
- Compare view is **sector-vs-sector** (not company-vs-company).
- Honor ClearyFi honesty rules: no good/bad favorability color; composite scores
  are provisional with open decomposition; no winner declared; A/B color is
  categorical identity only.
